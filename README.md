Petite précision pour lancer le bot depuis chez vous : 
1 : télécharger les fichier ici présent
2 : ouvrer votre terminal de commande
3 : mettez vous dans le dossier courant
4 : lancer le bot grace a la commance "npm run dev"
5 : il est possible que vous soyez contraint a télécharger npm
6 : pour que vos changement sois directement opérationnelle utilser la commande npm install nodemon --save-dev
7 : voila rien d'autre a dire amusez vous !
